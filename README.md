# SA-_105_TP_2_MARESIA_Enzo_FATET_Ga-tan_HOUNTONDJI_LIONEL
Exploitation des données d' un ballon sonde !
Dans ce projet on exploitera les données recues d' un ballon sonde envoyer par l' IUT d' auxerre.
On ne fera qu' exploiter les données on n' abordera pas le sujet de la conception et ou des contraints techniques.
Dans le dossier additionalfiles vous trouverez les scrypt adaptable et les images des courbes associé selon nos valeurs ainsi que le fichier .CSV que nous avons utiliser.
Pour exploiter ce scrypt il faudra impérativement avoir le fichier .CSV et le scrypt dans le même dossier/répertoire.
